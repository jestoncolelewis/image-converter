# Simple image converter
Inspired by 2022's CS50 course, I built this basic photo converter to work through the problem in Python instead of C for a web deployment with the processing happening server side. Decided to refactor the project into Javascript so it could run client side and be easier to deploy.

## Use
A simple UI are used to determine what mode the user wants to use in processing their image(s).
* Grayscale
* Sepia
* Mirror
* Reflect (Still troubleshooting)
* Scale
